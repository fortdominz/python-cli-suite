## Setup
Each project is self-contained in its own folder. Projects using external APIs 
require you to add your own credentials — look for placeholder values like 
`YOUR_API_KEY` in the source files.

## Note
API keys and credentials have been removed from all files. 
Replace placeholder values with your own before running.
